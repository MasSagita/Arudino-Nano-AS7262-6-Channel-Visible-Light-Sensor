# Arudino-Nano-AS7262-6-Channel-Visible-Light-Sensor
Glucose Content using Arduino Nano and AS7262 6 Channel Visible Light Sensor

# Hardware:
- Arduino Nano
- AS7262
- 2 Channel Relay (for AC heater and DC fan);  
- DS18B20
- DHT11
- Rotary Encoder Switch for Menu
